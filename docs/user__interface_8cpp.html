Detailed Description

User interface implementing a the services server.

Author
Eléa Papin
Version
1.0
Date
12/03/2022

Publishes to:
° /current_mode ° /move_base_goal

Services:
° /switch_mode ° /set_goal*

Description : This node handles the two services with which the user can communicate with the simulation. It updates the running mode via /switch_mode and broadcast it to the manual_node on /current_mode. It also relays the goal request sent by the user on /set_goal to the action client using /move_base/goal.
